# privello

Repositório privado — plataforma de acompanhantes com curadoria editorial, verificação manual e perfis auditáveis.

> Este repositório é de uso exclusivo do proprietário. Não é um projeto open source e não aceita contribuições externas.

---

**Stack:** Next.js 16 · TypeScript · Prisma · PostgreSQL · NextAuth v5 · Tailwind CSS

**Ambiente:** Requer `.env` com `DATABASE_URL` e `AUTH_SECRET`. Não há suporte para instalação externa.

## Documentação operacional

- Para subir o Postgres de dev, ver [docs/docker.md](./docs/docker.md).
- Para a lista canônica de variáveis de ambiente, ver [docs/env.md](./docs/env.md).
